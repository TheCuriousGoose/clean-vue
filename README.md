# clean-vue

This a clean template with all the Vue3 bloat cleaned up.

## Features
- No about page
- No premade styling
- No components
- Cleaned up router
- SCSS
- Typescript
- Pinia
- Prettier
- Eslint
- Vite

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
